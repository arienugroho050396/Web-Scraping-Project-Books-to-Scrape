
# 📚 Web Scraping Project — Books to Scrape

A simple but practical web scraping project to collect product data from a demo e-commerce website and transform it into a clean dataset.

This project is designed to simulate a **real-world data ingestion pipeline** (scraping → cleaning → storing).

![This is an image](https://github.com/arienugroho050396/Web-Scraping-Project-Books-to-Scrape/blob/main/image.png) 
---
 
## 🚀 Project Overview 

We scrape data from:
- http://books.toscrape.com/

And extract:
- 📖 Title
- 💰 Price
- ⭐ Rating
- 📦 Availability

Then:
- Clean the data
- Store it in a structured format (CSV / DataFrame)

---

## 🧰 Tech Stack

- Python
- requests
- BeautifulSoup
- pandas
